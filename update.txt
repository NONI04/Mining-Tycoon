===================================
  Mining Tycoon - 업데이트 노트
  버전: v0.2.0  (2026-05-24)
===================================

[ UI / 오른쪽 화면 ]
- Mining Tycoon, 업그레이드, 지상 창고 제목 폰트 크기 대폭 확대
- 자산 표시(💰 금액) 폰트 크기 확대 및 가운데 정렬
- 오른쪽 패널 너비 확대 (지상 창고 칸 여유 확보)
- 오른쪽 패널 모든 수치에 세 자리마다 쉼표(,) 추가
- 해금 버튼 가격에도 쉼표 추가

[ 업그레이드 ]
- 업그레이드 항목 사이에 구분선 추가
- 행운 업그레이드를 목록 맨 아래로 이동
- 업그레이드 설명을 "현재 상태 -> 업그레이드 후 상태" 형식으로 변경
  예) 층당 광부 총 2명 -> 3명 / 채굴 속도 +50% -> +100%
- MAX 달성 시 현재 레벨 표시 추가 (예: 강한 곡괭이 Lv.5 [MAX])
- 행운 업그레이드 재조정:
    Lv.1: 채굴량 2배  ($10,000,000)
    Lv.2: 채굴량 3배  ($30,000,000)
    Lv.3: 채굴량 5배  ($100,000,000)  ← MAX

[ 광부 / 채굴 ]
- 광부 추가 업그레이드 신설: 레벨당 각 층에 광부 1명 추가
- 광부가 광물을 캘 때마다 머리 위에 광물 이름 팝업 표시
- 채굴 시간이 광물 종류에 따라 다르게 적용 (비싼 광물일수록 오래 걸림)
  → 돌: 약 2.0초 / 다이아몬드: 약 6.0초 (약 3배 차이)
- 층 플랫폼 왼쪽으로 확장, 광부 위치 조정
- 광부 간격 조정 (6명 기준 오른쪽 끝 광부~상자 간격 = 광부 1명 크기)
- 광부 지상 강하 모션 제거 (층에 바로 등장)

[ 지상 창고 ]
- 광물마다 가로 구분선 추가
- 맨 아래에 전체 광물 '총 가격' 표시
- '모든 광물 판매' 버튼 추가
- 광물 이름 칸 너비 확대 (다이아몬드 등 긴 이름도 정렬 유지)
- 광부 수 표시 라벨 제거

[ 스크롤 ]
- 왼쪽 화면 스크롤 범위 제한:
    해금된 층까지만 스크롤 가능 (다음 해금 층이 화면 하단에 표시)
    전 층 해금 시 마지막 층(다이아몬드)이 화면 하단에 위치
- 스크롤 최대 시 하단 여백 2배로 확대
- 터치패드 두 손가락 스크롤 감도 조정 (마우스 휠 감도는 유지)

[ 기타 ]
- 테스트/리셋 버튼 왼쪽 화면 좌측 상단으로 이동
